# Open Foreground Boundary

Funded foreground in this repository:

- public benchmark runners;
- public-source adapters;
- pre-registration templates;
- tests and publication guards;
- source-lineage and hash workflow;
- sanitized evidence summaries;
- documentation for public review.

Excluded background:

- private production integrations;
- proprietary formulas or private algorithms;
- private commercial logic;
- platform-specific reference clients;
- native parity assets and golden-vector assets;
- raw proprietary data or private payload directories.

The open foreground is designed to be independently usable without private
background technology.
